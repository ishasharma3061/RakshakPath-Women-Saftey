# 🛡️ RAKSHAKPATH

### Guarded Routes for Every Woman

**Women Safety Application**

RakshakPath is a women safety application designed to provide immediate assistance during emergencies. The platform allows users to save trusted contacts and instantly send SOS alerts along with their location and personal information.

### ✨ Key Features

* 🔐 User Login and Registration
* 👩 Women's Profile Management
* ☎️ Save Two Trusted Emergency Contacts
* 🚨 SOS Emergency Button
* 📍 Instant Location Sharing
* 🌐 Multilingual Support
* 📱 Mobile Friendly Interface

### 📌 How It Works

1. User logs into the application.
2. Adds trusted emergency contacts.
3. In an emergency, presses the SOS button.
4. Current location and emergency information are sent instantly to trusted contacts.

### 🛠️ Tech Stack

* Frontend: React, Vite, Tailwind CSS
* Backend: Node.js, Express.js
* Database: Firebase Firestore
* Maps: Leaflet and OpenStreetMap

### 🤝 About

RakshakPath was developed as a hackathon project with the aim of enhancing women's safety through quick emergency response and accessible safety features.
